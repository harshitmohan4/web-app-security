# 🌐 Web Application Security Basics – OWASP Top 10

## Overview

This repository documents practical awareness of the OWASP Top 10 web application vulnerabilities as part of **Week 2 – Task 1** of the Cybersecurity Internship program. The focus is on understanding how vulnerabilities occur and how to prevent them — using safe, ethical demo environments only.

## Learning Objectives

- Understand all 10 OWASP Top 10 (2021) vulnerability categories
- Analyze at least 3 vulnerabilities in depth (how they occur + prevention)
- Explore safe demo examples using OWASP Juice Shop
- Develop a prevention-first, secure coding mindset
